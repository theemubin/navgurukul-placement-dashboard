## 🔗 Connection String Template

```
mongodb+srv://[USER]:[PASS]@[CLUSTER]/[DB]?retryWrites=true&w=majority
```

**Replace:**
- `[USER]` → `placement_user`
- `[PASS]` → Your password
- `[CLUSTER]` → From MongoDB (like `cluster0.abc1234.mongodb.net`)
- `[DB]` → `placement_database`

**Example:**
```
mongodb+srv://placement_user:MyPass123@cluster0.abc1234.mongodb.net/placement_dashboard?retryWrites=true&w=majority
```

## 🆘 Quick Troubleshooting

| Error | Fix |
|-------|-----|
| Connection refused | Check password, wait 1-2 min |
| Auth failed | Verify username & password |
| Can't find cluster | Refresh page, wait 5 min |
| String not found | Go to Cluster → Connect → Drivers → Node.js |
